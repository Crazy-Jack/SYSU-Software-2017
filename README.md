# IGEM2017-SYSU.Software

## Dependence

- python3.5
    + django 1.11.4
    + mysqlclient 1.3.12
    
- MySQL 5.7.19

## Note

1. Create igem2017/igem2017/mysql.cnf file with information to access your MySQL.

   For example:
   ```
   [client]
   database = DATABASENAME
   user = USER
   password = PASSWORD
   default-character-set = utf8
   ```
